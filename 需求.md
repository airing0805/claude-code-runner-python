# Claude Code Runner 开源策划方案

## 项目定位与目标

### 核心定位
**"让 Claude Code 变成可编程的 Web API 服务"**

一句话描述：将 Anthropic 的 Claude Code（AI 编程助手）封装为 REST API + Web 界面的开源服务，让团队可以灵活集成和使用 AI 编程能力。

### 差异化优势（相对于竞品）

| 竞品 | 定位 | Claude Code Runner 优势 |
|------|------|------------------------|
| copilot-api | Copilot 转 API | 直接封装 Claude Code，非代理 |
| anthropic-proxy | 格式转换 | 完整功能：用户系统、会话管理 |
| codex-openai-proxy | Codex 转 API | 可视化界面 + MCP 集成 |
| - | - | **技能系统 + 钩子机制** |

### 核心独特卖点（USP）

1. **唯一性**: 首个将 Claude Code SDK 直接 API 化的开源项目
2. **完整性**: 用户认证 + 会话管理 + API 密钥 + MCP + 技能 + 钩子
3. **灵活性**: 4 种权限模式、可配置钩子、自定义技能
4. **可视化**: 友好的 Web 界面，实时 SSE 流式输出
5. **可扩展**: 插件系统、MCP 服务器集成
