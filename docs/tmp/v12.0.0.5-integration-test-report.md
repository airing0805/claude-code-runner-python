### 4.3 Data Format Comparison

**v11 Format**:
```json
{
    "sessions": {
        "tab-1": { "id": "s1", "workingDir": "/path1", "timestamp": 1000 },
        "tab-2": { "id": "s2", "workingDir": "/path2", "timestamp": 2000 }
    },
    "activeTab": "tab-1"
}
```

**v12 Format**:
```json
{
    "sessions": [
        { "id": "s2", "workingDir": "/path2", "timestamp": 2000 },
        { "id": "s1", "workingDir": "/path1", "timestamp": 1000 }
    ],
    "currentSessionId": "s1",
    "workspaceHistory": ["/path2", "/path1"]
}
```
