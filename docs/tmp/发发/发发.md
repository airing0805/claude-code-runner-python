

# **OpenClaw** 多agent 执行时的架构流程

OpenClaw 在创建 agent 时，为每个agent创建独立的workspace


```mermaid
graph TD

    subgraph User_Interaction_Layer [用户交互层]

        User((用户))

        Channels[通信渠道<br/>WhatsApp/Telegram/飞书/Web UI]

    end

  

    subgraph OpenClaw_Core [OpenClaw 核心系统]

        Gateway[Gateway<br/>消息路由与协议转换]

        subgraph Agent_Coordination [Agent 协作编排层]

            Orchestrator{协作模式调度器<br/>线性/并行/辩论}

        end

  

        subgraph Agent_Runtime_Pool [Agent 运行时池]

            direction TB

            Agent_Coder[Coder Agent<br/>代码生成]

            Agent_Tester[Tester Agent<br/>测试验证]

            Agent_Reviewer[Reviewer Agent<br/>代码审查]

            Agent_Doc[Documenter Agent<br/>文档编写]

            Agent_Generic[Generic Agent<br/>通用任务]

        end

  

        subgraph Shared_Resources [共享资源层]

            Memory[(记忆系统<br/>Embedding/混合搜索)]

            Tools[工具库<br/>文件/浏览器/终端/API]

            Models[AI 模型池<br/>Anthropic/OpenAI/Google/本地]

        end

    end

  

    %% 关系连接

    User -->|发送指令 | Channels

    Channels -->|接收/发送消息 | Gateway

    Gateway -->|分发任务 | Orchestrator

    Orchestrator -->|分配子任务 | Agent_Coder

    Orchestrator -->|分配子任务 | Agent_Tester

    Orchestrator -->|分配子任务 | Agent_Reviewer

    Orchestrator -->|分配子任务 | Agent_Doc

    Orchestrator -->|分配子任务 | Agent_Generic

  

    %% Agent 间协作关系 (以软件开发流程为例)

    Agent_Coder -->|提交代码 | Agent_Tester

    Agent_Coder -->|请求审查 | Agent_Reviewer

    Agent_Tester -->|反馈结果 | Agent_Coder

    Agent_Reviewer -->|批准/驳回 | Agent_Coder

    Agent_Coder -->|请求文档 | Agent_Doc

    %% Agent 与共享资源交互

    Agent_Coder <-->|读写/检索 | Memory

    Agent_Tester <-->|读写/检索 | Memory

    Agent_Reviewer <-->|读写/检索 | Memory

    Agent_Doc <-->|读写/检索 | Memory

    Agent_Generic <-->|读写/检索 | Memory

  

    Agent_Coder -->|调用 | Tools

    Agent_Tester -->|调用 | Tools

    Agent_Reviewer -->|调用 | Tools

    Agent_Doc -->|调用 | Tools

    Agent_Generic -->|调用 | Tools

  

    Agent_Coder -->|推理 | Models

    Agent_Tester -->|推理 | Models

    Agent_Reviewer -->|推理 | Models

    Agent_Doc -->|推理 | Models

    Agent_Generic -->|推理 | Models

  

    %% 样式定义

    style User fill:#f9f,stroke:#333,stroke-width:2px

    style Gateway fill:#bbf,stroke:#333,stroke-width:2px

    style Orchestrator fill:#f96,stroke:#333,stroke-width:2px

    style Agent_Coder fill:#9f9,stroke:#333

    style Agent_Tester fill:#9f9,stroke:#333

    style Agent_Reviewer fill:#9f9,stroke:#333

    style Agent_Doc fill:#9f9,stroke:#333

    style Agent_Generic fill:#9f9,stroke:#333

    style Memory fill:#ff9,stroke:#333

    style Tools fill:#9ff,stroke:#333

    style Models fill:#f9f,stroke:#333
```


# Claude  Runner

## Claude  Runner 单项目

单独的项目在独立的目录运行，有系统级的配置，也支持项目级的配置，项目级的 skill 会覆盖系统级的 skill

```mermaid
graph TD

    subgraph User_Layer ["用户交互层"]
        User((用户))
        Client[Claude Code<br/>客户端]
    end

    subgraph ClaudeCode_Core ["Claude Code 核心系统"]

        subgraph MainAgent ["Main Agent"]
            MainAgentNode{主代理<br/>负责调度}
        end

        subgraph Agent_Registry ["Agent 注册中心 (配置)"]
            direction TB
            SysAgent[System Agent<br/>全局配置 ~/.claude/agents/]
            ProjAgent[Project Agent<br/>项目配置 ./.claude/agents/]

            subgraph SysAgents ["全局 Agent 列表"]
                direction TB
                subgraph SysAgent_Plan ["规划类"]
                    Planner[Planner<br/>实现规划]
                    Architect[Architect<br/>系统设计]
                end
                subgraph SysAgent_Dev ["开发类"]
                    TDDGuide[TDD Guide<br/>测试驱动]
                    CodeReviewer[Code Reviewer<br/>代码审查]
                    RefactorCleaner[Refactor Cleaner<br/>重构清理]
                end
                subgraph SysAgent_QA ["质量类"]
                    SecurityReviewer[Security Reviewer<br/>安全分析]
                    BuildResolver[Build Error<br/>构建修复]
                    E2ERunner[E2E Runner<br/>端到端测试]
                end
                DocUpdater[Doc Updater<br/>文档更新]
            end

            subgraph ProjAgents ["项目 Agent 列表"]
                ProjPlanner[项目规划<br/>Agent]
                ProjTester[测试<br/>Agent]
                ProjDebug[调试<br/>Agent]
                ProjDocs[文档<br/>Agent]
                ProjPlanner --> ProjTester --> ProjDebug --> ProjDocs
            end
        end

        subgraph Agent_Runtime_Pool ["Subagent 实例 (执行)"]
            Subagent1[Subagent 实例 1]
            Subagent2[Subagent 实例 2]
            Subagent3[Subagent 实例 3]
        end

        subgraph Shared_Resources ["共享资源层"]
            Memory[(记忆系统<br/>CLAUDE.md)]
            Tools[工具库<br/>File/Shell/Grep/MCP]
            Models[AI 模型池<br/>Opus/Sonnet]
        end

    end

    %% 流程
    User -->|输入任务| Client
    Client -->|会话| MainAgentNode

    %% Main Agent 加载配置
    MainAgentNode .->|加载配置| SysAgent
    MainAgentNode .->|加载配置优先| ProjAgent

    %% Agent 注册中心关联
    SysAgent -->|注册| SysAgents
    ProjAgent -->|注册| ProjAgents

    %% Main Agent 创建 Subagent 实例
    MainAgentNode ==>|创建实例| Subagent1
    MainAgentNode ==>|创建实例| Subagent2
    MainAgentNode ==>|创建实例| Subagent3

    %% Subagent 执行任务
    Subagent1 -->|调用| Tools
    Subagent2 -->|调用| Tools
    Subagent3 -->|调用| Tools

    Subagent1 <-->|读写| Memory
    Subagent2 <-->|读写| Memory
    Subagent3 <-->|读写| Memory

    Subagent1 -->|推理| Models
    Subagent2 -->|推理| Models
    Subagent3 -->|推理| Models

    %% 结果返回
    Subagent1 ==>|返回结果| MainAgentNode
    Subagent2 ==>|返回结果| MainAgentNode
    Subagent3 ==>|返回结果| MainAgentNode

    %% 样式定义
    style User fill:#f9f,stroke:#333
    style Client fill:#fef3c7,stroke:#333
    style MainAgentNode fill:#f96,stroke:#333,stroke-width:3px
    style SysAgent fill:#bbf,stroke:#333,stroke-dasharray:5 5
    style ProjAgent fill:#bbf,stroke:#333,stroke-dasharray:5 5
    style SysAgents fill:#e0e7ff,stroke:#333,stroke-dasharray:3 3
    style SysAgent_Plan fill:#dbeafe,stroke:#333,stroke-dasharray:3 3
    style SysAgent_Dev fill:#dbeafe,stroke:#333,stroke-dasharray:3 3
    style SysAgent_QA fill:#dbeafe,stroke:#333,stroke-dasharray:3 3
    style ProjAgents fill:#e0e7ff,stroke:#333,stroke-dasharray:3 3
    style Planner fill:#bfdbfe,stroke:#333
    style Architect fill:#bfdbfe,stroke:#333
    style TDDGuide fill:#bfdbfe,stroke:#333
    style CodeReviewer fill:#bfdbfe,stroke:#333
    style SecurityReviewer fill:#bfdbfe,stroke:#333
    style BuildResolver fill:#bfdbfe,stroke:#333
    style E2ERunner fill:#bfdbfe,stroke:#333
    style RefactorCleaner fill:#bfdbfe,stroke:#333
    style DocUpdater fill:#bfdbfe,stroke:#333
    style ProjPlanner fill:#dcfce7,stroke:#333
    style ProjTester fill:#dcfce7,stroke:#333
    style ProjDebug fill:#dcfce7,stroke:#333
    style ProjDocs fill:#dcfce7,stroke:#333
    style Subagent1 fill:#9f9,stroke:#333
    style Subagent2 fill:#9f9,stroke:#333
    style Subagent3 fill:#9f9,stroke:#333
    style Memory fill:#ff9,stroke:#333

    style Models fill:#fcf,stroke:#333
```

## Claude  Runner 多项目



```mermaid
graph TD

    subgraph User_Layer ["用户交互层"]
        User((用户))
        Client1[Claude Code<br/>客户端 1]
        Client2[Claude Code<br/>客户端 2]
    end

    subgraph ClaudeCode_Core ["Claude Code 核心"]

        subgraph Workspace1 ["工作空间 1 (项目 A)"]
            MainAgent1{Main Agent<br/>主代理}
            SubagentPool1[Subagent<br/>实例池]
            Memory1[(记忆系统<br/>CLAUDE.md)]
            ProjConfig1[项目配置<br/>.claude/agents/]
        end
        
		subgraph Global_Config ["全局配置"]
            SysConfig[System Agent<br/>~/.claude/agents/]
        end

        subgraph Shared_Resources ["共享资源层"]
            Tools[工具库<br/>File/Shell/Grep/MCP]
            Models[AI 模型池<br/>Opus/Sonnet]
        end
        subgraph WorkspaceN ["工作空间 N (项目 N...)"]
	        ProjConfigN[项目配置<br/>.claude/agents/]
            MainAgentN{Main Agent<br/>主代理}
            SubagentPoolN[Subagent<br/>实例池]
            MemoryN[(记忆系统<br/>CLAUDE.md)]
        end

        

    end

    %% 流程
    User -->|会话| Client1
    User -->|会话| Client2
    Client1 -->|工作| Workspace1
    Client2 -->|工作| WorkspaceN

    %% 工作空间内部
    MainAgent1 -->|创建| SubagentPool1
    MainAgent1 .->|加载| ProjConfig1
    MainAgent1 .->|加载| SysConfig
    SubagentPool1 -->|调用| Tools
    SubagentPool1 <-->|读写| Memory1
    SubagentPool1 -->|推理| Models

    MainAgentN -->|创建| SubagentPoolN
    MainAgentN .->|加载| ProjConfigN
    MainAgentN .->|加载| SysConfig
    SubagentPoolN -->|调用| Tools
    SubagentPoolN <-->|读写| MemoryN
    SubagentPoolN -->|推理| Models

    %% 样式定义
    style User fill:#f9f,stroke:#333
    style Client1 fill:#fef3c7,stroke:#333
    style Client2 fill:#fef3c7,stroke:#333
    style MainAgent1 fill:#f96,stroke:#333,stroke-width:3px
    style MainAgentN fill:#f96,stroke:#333,stroke-width:3px
    style SubagentPool1 fill:#9f9,stroke:#333
    style SubagentPoolN fill:#9f9,stroke:#333
    style Memory1 fill:#ff9,stroke:#333
    style MemoryN fill:#ff9,stroke:#333
    style ProjConfig1 fill:#bbf,stroke:#333,stroke-dasharray:5 5
    style ProjConfigN fill:#bbf,stroke:#333,stroke-dasharray:5 5
    style SysConfig fill:#bbf,stroke:#333
    style Tools fill:#9ff,stroke:#333
```

## Claude  Runner 多项目多agnet完整版

```mermaid
graph TD

    User((用户))

    CLI1[Claude Code 客户端 1]
    CLI2[Claude Code 客户端 2]

    subgraph Global_Agents [全局 Agents]
        SysAgent[System Agent<br/>全局可用]
    end

    subgraph Workspace_A [工作空间 A]
        MainAgentA{Main Agent A}

        subgraph Agent_Registry_A [Agent 注册中心 A]
            ProjAgentA[Project Agent A]
        end

        subgraph Agent_Runtime_Pool_A [Agent 运行时池 A]
            SubagentA1[Subagent A1]
            SubagentA2[Subagent A2]
        end

        subgraph Resources_A [资源层 A]
            MemoryA[(CLAUDE.md)]
            ToolsA[工具库]
            ModelsA[AI 模型]
            FileSystemA[(文件系统 A)]
        end
    end

    subgraph Workspace_B [工作空间 B]
        MainAgentB{Main Agent B}

        subgraph Agent_Registry_B [Agent 注册中心 B]
            ProjAgentB[Project Agent B]
        end

        subgraph Agent_Runtime_Pool_B [Agent 运行时池 B]
            SubagentB1[Subagent B1]
            SubagentB2[Subagent B2]
        end

        subgraph Resources_B [资源层 B]
            MemoryB[(CLAUDE.md)]
            ToolsB[工具库]
            ModelsB[AI 模型]
            FileSystemB[(文件系统 B)]
        end
    end

    %% 用户使用多个客户端
    User -->|使用| CLI1
    User -->|使用| CLI2

    %% 客户端与工作空间一对一连接
    CLI1 -->|cd /workspace/A| MainAgentA
    CLI2 -->|cd /workspace/B| MainAgentB

    %% Main Agent 扫描全局 System Agent
    MainAgentA -.->|扫描| SysAgent
    MainAgentB -.->|扫描| SysAgent

    %% Main Agent 扫描项目 Agent
    MainAgentA -.->|扫描优先| ProjAgentA
    MainAgentB -.->|扫描优先| ProjAgentB

    %% 工作空间A内部流程
    MainAgentA ==>|委派| SubagentA1
    MainAgentA ==>|委派| SubagentA2
    SubagentA1 -->|调用| ToolsA
    SubagentA2 -->|调用| ToolsA
    SubagentA1 <-->|读写| MemoryA
    SubagentA2 <-->|读写| MemoryA
    SubagentA1 -->|推理| ModelsA
    SubagentA2 -->|推理| ModelsA
    SubagentA1 -->|操作| FileSystemA
    SubagentA2 -->|操作| FileSystemA
    SubagentA1 ==>|报告| MainAgentA
    SubagentA2 ==>|报告| MainAgentA

    %% 工作空间B内部流程
    MainAgentB ==>|委派| SubagentB1
    MainAgentB ==>|委派| SubagentB2
    SubagentB1 -->|调用| ToolsB
    SubagentB2 -->|调用| ToolsB
    SubagentB1 <-->|读写| MemoryB
    SubagentB2 <-->|读写| MemoryB
    SubagentB1 -->|推理| ModelsB
    SubagentB2 -->|推理| ModelsB
    SubagentB1 -->|操作| FileSystemB
    SubagentB2 -->|操作| FileSystemB
    SubagentB1 ==>|报告| MainAgentB
    SubagentB2 ==>|报告| MainAgentB

    %% 样式定义
    style User fill:#f9f,stroke:#333

    style CLI1 fill:#fef3c7,stroke:#333
    style CLI2 fill:#fef3c7,stroke:#333

    style SysAgent fill:#bbf,stroke:#333,stroke-dasharray:5 5

    style MainAgentA fill:#f96,stroke:#333,stroke-width:2px
    style MainAgentB fill:#f96,stroke:#333,stroke-width:2px

    style Workspace_A fill:#e1f5fe,stroke:#333
    style Workspace_B fill:#f3e5f5,stroke:#333
```
