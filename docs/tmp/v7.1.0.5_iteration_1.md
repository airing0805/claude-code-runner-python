### ❌ 需要修复的问题

#### 问题 1: 缺少虚拟滚动实现 (严重程度: 高)

**问题描述**:
- logViewer.js 中 renderLogs() 方法直接渲染所有日志条目
- 当日志数量很大时（如数千条），会导致性能问题
- 需求文档明确要求使用虚拟滚动优化

**当前代码** (logViewer.js:231-261):
```javascript
renderLogs() {
    // 直接渲染所有日志，没有虚拟滚动
    this.state.logs.forEach((log, index) => {
        // ... 渲染每一行
    });
}
```

**修复建议**:
- 参考项目中其他模块（如 history.js）中的虚拟滚动实现
- 实现 VirtualList 或类似机制
- 只渲染可见区域的日志条目
