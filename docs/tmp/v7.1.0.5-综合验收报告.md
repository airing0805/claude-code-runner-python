## 5. 发现问题
### 5.1 敏感信息脱敏功能问题
- **位置**: `app/services/questions.py` - `mask_sensitive_info` 函数
- **问题**: `password=secret123` 格式的密码无法正确脱敏
- **原因**: 现有正则只匹配 `password="value"` 格式，不匹配 `password=value` 格式
- **影响**: 中等 - 影响安全性和用户隐私
- **修复方案**: 添加不带引号的 password 匹配模式

  ```python
  (r"password=[^\\s==[^\\s]+", "[PASSWORD]"),  # 带引号格式
  (r"password=[a-zA-Z0-9]+", "[PASSWORD]"),  # 无空格/引号格式
  ```
