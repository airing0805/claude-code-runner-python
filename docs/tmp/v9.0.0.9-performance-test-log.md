## 测试用例

### 1. API 响应时间测试

#### 1.1 首页加载时间测试
- **测试文件**: `tests/test_performance.py::TestAPIResponseTime::test_index_page_load_time`
- **测试内容**: 测试首页 (/) 加载响应时间
- **预期**: 响应时间 < 500ms

#### 1.2 静态文件加载时间测试
- **测试文件**: `tests/test_performance.py::TestAPIResponseTime::test_static_file_load_time`
- **测试内容**: 测试 JavaScript 和 CSS 文件加载时间
- **预期**: 响应时间 < 200ms

#### 1.3 健康检查端点测试
- **测试文件**: `tests/test_performance.py::TestAPIResponseTime::test_api_health_check`
- **测试内容**: 测试 /api/status 端点响应时间
- **预期**: 响应时间 < 100ms

#### 1.4 任务 API 响应时间测试
- **测试文件**: `tests/test_performance.py::TestAPIResponseTime::test_task_api_response_time`
- **测试内容**: 测试 POST /api/task 端点响应时间
- **预期**: 响应时间包含完整任务执行

## 已创建的测试文件

### 1. tests/test_performance.py
完整的 pytest 性能测试套件，包含以下测试类：

- **TestAPIResponseTime** - API 响应时间测试
  - test_index_page_load_time - 首页加载时间测试
  - test_static_file_load_time - 静态文件加载时间测试
  - test_api_health_check - 健康检查端点测试
  - test_task_api_response_time - 任务 API 响应时间测试

- **TestSSEMessageLatency** - SSE 消息延迟测试
  - test_sse_connection_time - SSE 连接建立时间测试
  - test_sse_message_intervals - SSE 消息间隔测试

- **TestLargeMessageHandling** - 大消息量处理测试
  - test_large_prompt_handling - 大 prompt 处理测试
  - test_high_message_volume - 高消息量处理测试

- **TestConcurrentRequests** - 并发请求测试
  - test_concurrent_task_requests - 并发任务请求测试

### 2. tests/manual_perf_test.py
手动性能测试脚本，可在服务器运行时快速执行性能测试。

使用方式:
```bash
python tests/manual_perf_test.py
```
