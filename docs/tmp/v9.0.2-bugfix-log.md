### 1. 修复消息过滤问题

#### 1.1 修复空内容消息丢失问题

**问题描述**: 当消息内容为空时，该消息会被丢弃，导致消息链不完整。

**修复文件**: `app/routers/session.py`

**修复位置**:
- `parse_content_blocks()` 函数 (第 434-532 行)
- `get_session_messages()` 函数 (第 535-622 行)

**修复方案**:
- 修改 `parse_content_blocks()` 函数，即使内容为空也保留消息结构
- 移除 `get_session_messages()` 中 `if content_blocks:` 的检查，始终添加消息

**关键代码变更**:
```python
# parse_content_blocks() 修改
# 之前: 空内容返回空数组
# 之后: 保留空内容块用于消息链完整性

# 处理字符串类型的 content
if isinstance(message_content, str):
    # v9.0.2: 即使内容为空字符串，也保留消息结构
    blocks.append({
        "type": "text",
        "text": message_content,
    })
    return blocks

# 添加空内容块的兜底处理
if not blocks:
    blocks.append({
        "type": "text",
        "text": "",
    })
```

```python
# get_session_messages() 修改
# 之前:
# if content_blocks:
#     messages.append({...})

# 之后: v9.0.2: 始终添加消息，即使内容为空
messages.append({...})
```

**修复位置**: `parse_content_blocks()` 函数

**修复方案**:

**关键代码变更**:
```python
# 第一遍：收集 tool_use 的 ID 到名称的映射（包括空名称的情况）
tool_use_map: dict[str, str] = {}
for item in message_content:
```

- `updateContinueConversationState()` 函数 (第 291-326 行)
- `handleContinueConversationChange()` 函数 (第 330-362 行)

**修复方案**:
- 在 `getLatestSessionId()` 函数中添加 `workingDir` 参数
- 在 API 请求中添加 `working_dir` 参数

**关键代码变更**:
```javascript
// 之前
async getLatestSessionId() {
    const response = await fetch('/api/sessions?limit=1');
    ...
}

// 之后
async getLatestSessionId(workingDir = null) {
    const dir = workingDir || this.workingDirInput?.value || '.';
```

### 3. 大文件会话加载 - 分页加载

**问题描述**: 大文件会话一次性加载所有消息，导致加载缓慢、页面卡顿。

**修复文件**:
- 后端: `app/routers/session.py`
- 前端: `web/static/modules/history.js`

**修复位置**:
- 后端: `get_session_messages()` 函数
- 前端: `continueSession()` 函数及相关辅助函数

**修复方案**:
1. 后端添加分页参数支持 (`page`, `limit`)
2. 返回分页元数据 (`pagination` 对象)
3. 前端实现分页加载逻辑

**关键代码变更**:

后端 API:
```python
@router.get("/sessions/{session_id}/messages")
async def get_session_messages(
    session_id: str,
    page: int = Query(1, ge=1, description="页码"),
    limit: int = Query(100, ge=1, le=500, description="每页数量"),
):
    # v9.0.2: 添加分页支持
    # 返回分页信息
    return {
        "session_id": session_id,
        "project_path": final_path,
        "messages": paginated_messages,
        "pagination": {
            "page": page,
            "limit": limit,
            "total": total,
            "pages": pages,
            "has_more": page < pages,
        }
    }
```

前端加载:
```javascript

// v9.0.2: 生成错误详情
let errorDetail = '网络连接中断';
if (error.message) {
    if (error.message.includes('network') || error.message.includes('fetch')) {
        errorDetail = '网络连接失败，请检查网络';
    } else if (error.message.includes('timeout')) {
        errorDetail = '连接超时，请稍后重试';
    } else if (error.message.includes('abort')) {
        errorDetail = '请求被中止';
    } else {
        errorDetail = '服务器连接中断';
    }
}

// 重连时显示进度
this.addMessage(runner, 'info', `${errorDetail}，正在尝试重连 (${this._retryCount}/${SSE_CONFIG.MAX_RETRIES})...`);
```
