# PM2任务配置

> Claude Code Runner 多智能体项目修复计划的 PM2 配置

---

## 配置说明

### 执行时间

- **启动时间**：第0分钟开始执行
- **每轮时长**：20分钟
- **总时长**：5小时（300分钟）

### Cron表达式说明

| 模式 | 含义 | 示例 |
|------|------|------|
| `*/N * * * *` | 每N分钟执行一次 | `*/20 * * * *` 每20分钟执行 |
| `0/20 0 * * *` | 第0小时的每20分钟执行一次 | 00:00, 00:20, 00:40 |
| `10/20 0 * * *` | 第0小时的第10分钟开始每20分钟执行 | 00:10, 00:30, 00:50 |

## 注意事项

1. **顺序执行**：确保执行和评审交替进行
2. **依赖管理**：后阶段依赖前阶段输出
3. **异常处理**：记录问题到日志，继续执行
4. **结果保存**：每轮结果保存到指定文件
5. **超时控制**：每任务最长20分钟

## 配置安装

```bash
# 方式1：手动添加到PM2配置文件
# 编辑 ~/.claude/scheduler/scheduled-tasks.json
# 将上述JSON配置添加到数组中

# 方式2：使用PM2命令添加
/scheduler:schedule-add
# 然后手动编辑每个任务的配置
```
