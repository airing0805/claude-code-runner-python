### 问题 2：内存存储无持久化

**严重程度**: 中 (非阻塞)
**影响**: 服务重启后用户数据丢失

**说明**:
- 用户数据存储在 `_users_db` 内存字典
- API Key 存储在 `_api_keys_db` 内存字典
- Token 黑名单存储在 `_token_blacklist` 内存集合

**已记录**: 需求文档已说明"生产环境应使用数据库"

---

### 问题 3：SECRET_KEY 默认值不安全

**严重程度**: 中
**文件**: app/auth/core.py:22

```python
SECRET_KEY = os.getenv("SECRET_KEY", os.urandom(32).hex())
```

**问题**: 每次启动生成新的随机密钥会导致所有已有 Token 失效

**修复建议**: 启动时检查 SECRET_KEY 环境变量，如未设置则警告并退出

---

### 问题 4：限流器无并发锁

**严重程度**: 低
**文件**: app/auth/rate_limiter.py

**问题**: 在高并发场景下可能存在竞态条件

**说明**: 当前实现使用 defaultdict，在并发访问时可能出现计数不准确
