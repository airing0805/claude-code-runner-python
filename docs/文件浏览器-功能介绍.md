# 文件浏览器功能介绍

本文档介绍 Claude Code Runner v11.0.0 版本新增的文件浏览器功能。

## 功能概述

文件浏览器让用户能够通过浏览器直观地浏览项目文件结构、预览文件内容。系统充分利用 Claude Code 的文件操作能力，实现目录树展示、文件预览、文件搜索等功能。

## 核心特性

| 特性 | 说明 |
|------|------|
| 目录树浏览 | 树形展示项目文件结构，支持展开/折叠 |
| 文件预览 | 查看文件内容，支持大文件分页 |
| 文件搜索 | 支持 Glob 模式搜索，快速定位文件 |
| 面包屑导航 | 显示当前路径，支持点击跳转 |
| 安全控制 | 路径验证、敏感文件保护 |

## 使用方法

### 访问文件浏览器

1. 在左侧导航栏中找到「文件浏览器」菜单项（如果没有，可能是作为独立对话框或面板形式存在）
2. 点击进入文件浏览器视图

### 目录树操作

#### 展开/折叠目录

- 点击目录左侧的箭头图标展开目录
- 再次点击可折叠目录
- 目录显示在前，文件显示在后

#### 选择文件

- 点击文件即可在右侧预览文件内容
- 当前选中的文件会有高亮显示

### 文件预览

#### 查看文件内容

- 选中文件后，右侧面板显示文件内容
- 文件名、路径、大小信息显示在预览区顶部
- 支持代码高亮显示

#### 分页浏览

- 大文件自动分页显示
- 底部显示分页信息：「第 1-500 行，共 1500 行」
- 点击「上一页/下一页」或直接输入页码跳转

### 文件搜索

#### 使用搜索框

1. 在搜索框中输入 Glob 模式
2. 点击「搜索」按钮或按 Enter 键
3. 搜索结果显示在下拉列表或新面板中

#### Glob 模式示例

| 模式 | 说明 | 示例 |
|------|------|------|
| `*.py` | 所有 Python 文件 | main.py, utils.py |
| `**/*.py` | 递归所有 Python 文件 | src/main.py, tests/test.py |
| `src/*` | src 目录下所有文件 | src/main.py |
| `*.{py,js}` | Python 和 JavaScript 文件 | main.py, app.js |

### 面包屑导航

- 顶部显示当前路径
- 点击路径中的任意目录可快速跳转
- 例如：`项目根目录 > src > utils > main.py`

## 安全控制

### 路径验证

系统会验证所有文件访问请求：

- 只能访问工作目录内的文件
- 禁止使用 `..` 跳出目录
- 禁止访问系统目录

### 敏感文件保护

以下文件类型被禁止访问：

| 文件类型 | 说明 |
|----------|------|
| .env | 环境变量文件 |
| .pem | 私钥文件 |
| .key | 密钥文件 |
| .secret | 密钥文件 |
| .password | 密码文件 |

### 访问限制

| 限制项 | 最大值 |
|--------|--------|
| 目录树深度 | 5 层 |
| 文件大小 | 10 MB |
| 读取行数 | 1000 行 |
| 搜索结果 | 500 个 |

## 移动端适配

文件浏览器功能支持响应式布局：

- 768px 断点：调整布局为单栏或双栏
- 480px 断点：优化小屏幕显示
- 触摸友好：按钮和交互元素适合触摸操作

## 常见问题

### Q: 为什么无法访问某些文件？

A: 系统可能因为以下原因拒绝访问：
- 文件超出工作目录范围
- 文件是敏感文件（.env, .pem 等）
- 文件过大（超过 10MB）
- 目录深度超过限制（5 层）

### Q: 大文件如何预览？

A: 大文件会自动分页显示，每页最多 1000 行。您可以使用底部分页控件浏览不同部分。

### Q: 搜索结果太多怎么办？

A: 使用更具体的 Glob 模式来缩小搜索范围。例如，使用 `src/**/*.py` 而不是 `**/*.py`。

### Q: 文件浏览器在哪里打开？

A: 文件浏览器可以通过以下方式打开：
1. 从主导航栏进入
2. 在任务执行过程中通过相关操作触发
