## 修复内容

在 `app/routers/session_manager.py` 的 `SessionStateMachine.TRANSITIONS` 中添加：

```python
# 等待状态出错（如超时）
(SessionStateEnum.WAITING, "task_error"): SessionStateEnum.FAILED,
# 等待状态超时
(SessionStateEnum.WAITING, "timeout"): SessionStateEnum.FAILED,
```

这确保当会话处于 WAITING 状态时，如果发生错误或超时，可以正确转换到 FAILED 状态。
