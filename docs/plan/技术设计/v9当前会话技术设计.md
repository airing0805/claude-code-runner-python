### 1.4 核心设计原则

**SOLID 原则应用**：

1. **单一职责 (SRP)**：
   - `SessionManager` 只负责会话状态管理，不涉及业务逻辑
   - `ClaudeCodeClient` 只负责与 SDK 交互，不处理 HTTP 请求
   - 每个 API 路由文件专注于单一资源领域

2. **开闭原则 (OCP)**：
   - 通过抽象接口 `SessionManager` 支持扩展新功能
   - 消息类型通过枚举定义，便于扩展新消息类型

3. **依赖倒置 (DIP)**：
   - API 层依赖抽象的会话管理接口，不直接依赖具体实现
   - 使用 Pydantic 模型进行数据校验，与具体业务逻辑解耦

## 2. 会话管理设计

### 2.1 SessionManager 模块设计

#### 2.1.1 核心职责

`SessionManager` 负责管理活跃会话的生命周期，包括：
- 创建和销毁会话
- 维护会话状态（等待中、执行中、已完成）
- 提供并发安全的会话访问
- 会话过期清理

#### 2.1.2 类设计

```python
class SessionManager:
    """会话管理器"""

    def __init__(self):
        self._sessions: dict[str, SessionState] = {}
        self._lock = asyncio.Lock()

    async def create_session(self, session_id: str, client: ClaudeCodeClient) -> SessionState:
        """创建新会话"""

    async def get_session(self, session_id: str) -> Optional[SessionState]:
        """获取会话状态"""

    async def get_session_info(self, session_id: str) -> Optional[SessionInfo]:
        """获取会话信息（不含客户端）"""

    async def list_sessions(self) -> list[SessionInfo]:
        """列出所有会话"""

    async def remove_session(self, session_id: str) -> None:
        """移除会话"""

    async def set_waiting(self, session_id: str, waiting: bool) -> None:
        """设置等待状态"""

    async def cleanup_old_sessions(self, max_age_seconds: float = 14400) -> int:
        """清理过期会话"""
```

### 2.2 会话状态转换机

#### 2.2.1 状态定义

| 状态 | 说明 | 触发条件 |
|------|------|----------|
| `idle` | 初始状态 | 会话创建前 |
| `running` | 执行中 | 任务开始执行 |
| `waiting` | 等待回答 | SDK 发出 AskUserQuestion |
| `completed` | 已完成 | 任务执行完成 |
| `error` | 错误 | 执行过程中发生异常 |

```

              │            │             │
              │            │ 答案提交     │
              │            ▼             │
              │     ┌─────────────┐       │

              │     └──────┬──────┘
              │            │

              │            ▼
              │     ┌─────────────┐

         │   error    │

              │
              │ remove_session()
              ▼
         ┌─────────────┐
         │  removed   │
         └─────────────┘
```

#### 2.2.3 状态转换规则

| 从状态 | 到状态 | 触发条件 | 动作 |
|--------|--------|----------|------|
| idle | running | 创建会话 | 初始化 SessionState |
| running | waiting | SDK 返回 ask_user_question | 设置 is_waiting=True |
| waiting | running | 提交答案 | 设置 is_waiting=False |
| running | completed | SDK 返回 complete | 标记会话完成 |
| running | error | 异常发生 | 标记错误，保留会话供调试 |
| any | removed | 显式删除或超时 | 清理会话资源 |

#### 2.3.3 过期会话清理

```python
async def cleanup_old_sessions(self, max_age_seconds: float = 14400) -> int:
    """清理过期会话"""
    current_time = time.time()
    to_remove = []

    async with self._lock:
        for session_id, state in self._sessions.items():
            if current_time - state.created_at > max_age_seconds:
                to_remove.append(session_id)

        for session_id in to_remove:
            del self._sessions[session_id]

    return len(to_remove)
```

|------|------|----------|
| 会话元数据缓存 | `MetadataCache` | 基于文件修改时间 (mtime) |
| 项目列表缓存 | `MetadataCache` | 按项目独立缓存 |

```python
class MetadataCache:
    """元数据缓存 - 基于文件修改时间"""

    def get(self, session_file: Path) -> dict | None:
        """获取缓存（检查 mtime）"""
        cached = self._cache.get(session_file.name)
        if cached and cached.get("mtime") == session_file.stat().st_mtime:
            return cached
        return None
```
