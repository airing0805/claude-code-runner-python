#### 数据模型定义

**SessionInfo（会话信息）**: 包含会话唯一标识、客户端实例、状态、等待标志、问题 ID、时间戳等字段

**SessionStatus（会话状态）**: idle（空闲）、executing（执行中）、waiting（等待答案）、completed（已完成）、error（错误）

**Message（消息）**: 包含消息类型、内容、时间戳、工具相关字段、问答相关字段、错误字段、元数据

**MessageType（消息类型）**: text、thinking、tool_use、tool_result、error、complete、info、ask_user_question

**UserQuestion（用户问答）**: 包含问题 ID、标题、描述、选项列表、多选标志等

**QuestionOption（问题选项）**: 包含选项值、标签、描述、预览内容

**QuestionAnswer（答案）**: 包含会话 ID、问题 ID、答案内容、追问答案

#### API 设计

**POST /api/task（同步执行）**: 请求包含 prompt（必填）、working_dir、tools、permission_mode、resume、continue_conversation 等字段；响应包含 success、message、session_id、result、error

**POST /api/task/stream（流式执行）**: 请求同 TaskRequest，返回 SSE 流，事件包含 type、content、timestamp、tool_name、tool_input、tool_use_id、error_type、metadata

**POST /api/task/stop（停止任务）**: 请求包含 session_id，响应包含 success、message

#### 错误码定义

| 错误码 | HTTP 状态码 | 说明 | 用户提示 |
|--------|-------------|------|----------|
| `INVALID_PROMPT` | 400 | 任务描述为空或过长 | "任务描述不能为空或超过 10000 字符" |
| `INVALID_WORKING_DIR` | 400 | 工作目录不存在 | "工作目录不存在或无权限访问" |

| `SESSION_NOT_WAITING` | 400 | 会话不在等待状态 | "当前会话不在等待回答状态" |

| `INVALID_ANSWER` | 400 | 答案格式无效 | "答案格式不正确，请重新选择" |
| `TASK_INTERRUPTED` | 400 | 任务已被中断 | "任务已被用户停止" |
| `ANSWER_TIMEOUT` | 408 | 答案提交超时 | "提交超时，请重新尝试" |
| `RATE_LIMIT_EXCEEDED` | 429 | 超过限流 | "请求过于频繁，请稍后再试" |
| `INTERNAL_ERROR` | 500 | 服务器内部错误 | "服务内部错误，请稍后重试" |

#### 任务清单

- [ ] 编写会话创建流程设计
  - [ ] 定义会话 ID 生成规则
  - [ ] 设计会话初始化流程
  - [ ] 设计 continue_conversation 处理

- [ ] 编写会话状态转换设计
  - [ ] 定义状态转换矩阵
  - [ ] 设计状态转换触发条件
  - [ ] 设计状态转换回调机制

- [ ] 编写会话清理流程设计
  - [ ] 定义清理触发条件
  - [ ] 设计资源回收机制
  - [ ] 设计持久化清理策略

#### 生命周期设计

**会话创建流程**: 检查 continue_conversation 获取或生成会话 ID，创建 SessionInfo，初始化 ClaudeCodeClient，注册到 SessionManager，返回会话信息

**状态转换矩阵**: idle→executing（用户发送任务）、executing→waiting（收到 AskUserQuestion）、waiting→executing（收到用户答案）、executing→completed/error（任务完成/失败）、any→terminated（停止/超时）、terminated→cleaned（清理完成）

**清理流程**: 停止任务执行、关闭 SSE 连接、释放客户端资源、从 SessionManager 移除、持久化会话数据（如需要）
