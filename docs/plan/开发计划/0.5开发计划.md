# Claude Code Runner 0.5 版本开发计划

## 项目背景

基于对参考项目 `claude-run 参考项目` 、`claude-run 参考项目\web\components` (React + Tailwind CSS) 的分析，优化当前项目的页面渲染和交互体验。

**技术栈对比：**
- 参考项目：React + Tailwind CSS + lucide-react 图标
- 当前项目：原生 JavaScript + 模块化 CSS

**当前项目前端结构：**
```
web/
├── templates/index.html      # 单页面模板
├── static/
│   ├── style.css             # 主样式入口
│   ├── css/                  # 模块化 CSS
│   │   ├── variables.css     # CSS 变量
│   │   ├── layout.css        # 布局样式
│   │   ├── components.css    # 组件样式
│   │   ├── tools.css         # 工具样式
│   │   └── responsive.css    # 响应式
│   ├── app.js                # 主入口
│   ├── components/           # UI 组件
│   └── modules/              # 功能模块
```

## 技术选型

| 组件/功能 | 选型方案 | 说明 |
|----------|---------|------|
| Markdown 渲染 | **marked.js (CDN)** | 体积小 (~40KB)，支持 GFM，适合原生 JS 项目 |
| Diff 可视化 | **jsdiff (CDN)** | 功能完整，支持多种 diff 格式 |
| 工具图标 | **Unicode emoji** | 无依赖、跨浏览器兼容，如 📄✏️⌨️🔍 |
| 代码高亮 | 可选 highlight.js (CDN) | 如需语法高亮可引入 |

**CDN 引入方式 (在 index.html 中添加)：**
```html
<!-- Markdown 渲染 -->
<script src="https://cdn.jsdelivr.net/npm/marked/marked.min.js"></script>

<!-- Diff 库 -->
<script src="https://cdn.jsdelivr.net/npm/diff@5.1.0/dist/diff.min.js"></script>

<!-- 可选：代码高亮 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/highlight.js@11.9.0/styles/github-dark.min.css">
<script src="https://cdn.jsdelivr.net/npm/highlight.js@11.9.0/lib/core.min.js"></script>
```
