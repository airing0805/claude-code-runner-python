## v11.0.0 - 项目文件浏览器核心功能

### 功能概述

通过浏览器直观地浏览项目文件结构、预览文件内容。充分利用 Claude Code 的文件操作能力，实现目录树展示、文件预览、文件搜索等功能。

## API 设计

### 1. 获取目录文件树

```
GET /api/files/tree?path={path}&depth={depth}&include_hidden={bool}

Response:
{
  "path": "/home/user/project",
  "name": "project",
  "type": "directory",
  "children": [
    {
      "name": "src",
      "type": "directory",
      "children": [...]
    }
  ],
  "metadata": {"total_files": 25, "total_dirs": 8}
}
```

### 2. 读取文件内容

```
GET /api/files/read?path={path}&start_line={n}&limit={n}

Response:
{
  "path": "/home/user/project/src/main.py",
  "name": "main.py",
  "size": 2345,
  "totalLines": 89,
  "content": "def main():\n    ...",
  "truncated": false,
  "hasMore": false
}
```

### 3. 搜索文件

```
GET /api/files/search?pattern={pattern}&path={path}&limit={n}

Response:
{
  "pattern": "**/*.py",
  "matches": [...],
  "total": 25,
  "truncated": false
}
```

### 4. 获取文件信息

```
GET /api/files/info?path={path}

Response:
{
  "path": "/home/user/project/src/main.py",
  "name": "main.py",
  "type": "file",
  "size": 2345,
  "sizeFormatted": "2.3 KB",
  "extension": ".py",
  "mimeType": "text/x-python",
  "modified": "2026-02-19T10:30:00Z"
}
```

### 5. Glob 模式查询

```
POST /api/files/glob
{
  "pattern": "src/**/*.py",
  "path": "/project",
  "limit": 200
}

Response:
{
  "pattern": "src/**/*.py",
  "matches": ["src/main.py", "src/utils.py"],
  "count": 3
}
```

## UI 设计

### 布局结构

```
+------------------+------------------------+
|   文件树         |   文件预览             |
|   (左侧)         |   (右侧)               |
|                  |                        |
|  - src/          |  1 | def main():      |
|    - main.py     |  2 |     print()      |
|    - utils.py    |  3 |     ...          |
|  - README.md     |                        |
|                  |                        |
+------------------+------------------------+
|  搜索框: [*.py        ] [搜索]           |
+--------------------------------------------+
```

### 组件设计

1. **文件树组件**
   - 树形结构，支持展开/折叠
   - 文件夹在前，文件在后
   - 支持右键菜单（打开、复制路径）

2. **文件预览组件**
   - 行号显示
   - 代码高亮（使用 Prism.js 或 highlight.js）
   - 大文件分页加载

3. **面包屑导航**
   - 显示当前路径
   - 支持点击跳转

4. **搜索框**
   - 支持 Glob 模式（**/*.py, src/*.js）
   - 实时搜索结果

## 安全考虑

| 安全措施 | 描述 |
|----------|------|
| 路径验证 | 防止目录遍历攻击（../../etc） |
| 大文件限制 | 单次最多读取 1000 行 |
| 敏感文件排除 | .env, *.pem, *.key 等 |
| 工作目录限制 | 必须在允许的工作目录范围内 |
