## 1. 需求概述

### 1.1 项目背景

v9 版本是项目的"当前会话"功能开发计划，旨在为用户提供与 Claude Code CLI 相似的交互界面，支持任务执行、会话管理和历史记录查看。该功能是 Claude Code Runner 的核心功能模块。

### 1.2 功能清单与优先级

根据现有需求文档 [当前会话-主需求.md](../需求文档/当前会话功能/当前会话-主需求.md)，功能清单如下：

| 功能模块 | 需求编号 | 优先级 | 状态 | 说明 |
|---------|---------|--------|------|------|
| 基本任务执行 | FR-001 | P0 | ✅ 已实现 | 同步/流式执行 |
| 会话历史查询 | FR-002 | P0 | ✅ 已实现 | 项目、会话列表 |
| 会话恢复 | FR-003 | P0 | ✅ 已实现 | resume 参数 |
| 问答交互 | FR-004 | P2 | ✅ 已实现 | AskUserQuestion 支持 |
| 消息展开/收起 | FR-005 | P1 | ⏸ 待实现 | 用户体验优化 |
| 标签页管理 | FR-006 | P2 | ⏸ 待实现 | 拖拽排序等 |

### 2.3 FR-003: 会话恢复需求评审

#### 2.3.1 需求完整性检查

| 需求项 | 实现状态 | 说明 |
|--------|---------|------|
| resume 参数支持 | ✅ 已实现 | 通过 session_id 恢复会话 |
| continue_conversation | ⚠️ 部分实现 | 前端逻辑需完善 |
| 最近会话 ID 获取 | ⚠️ 部分实现 | 需前端实现 |
| 会话文件验证 | ✅ 已实现 | check_session_file_valid() |

#### 2.3.2 会话恢复流程验证

根据 [当前会话-继续会话-需求.md](../需求文档/当前会话功能/当前会话-继续会话-需求.md)：

```
会话 ID 确定逻辑：
1. new_session=True → 创建新会话
2. resume 参数有效 → 检查内存会话和文件有效性
3. 创建新会话
```

**后端实现** (app/routers/task.py):
- 会话文件有效性检查已实现
- 内存会话管理已实现
- 支持恢复服务重启前的会话

**前端实现** (web/static/modules/task.js):
- 继续会话复选框逻辑已实现
- resumeInput 值处理已实现
- 会话 ID 自动填充已实现

#### 2.3.3 需求评估结论

**需求完整性**: ⚠️ 基本完整，需完善

**已实现**:
- resume 参数支持完整
- 会话恢复功能正常工作
- 会话文件验证机制完善

**待完善**:
- 最近会话 ID 获取机制的前端实现（根据需求文档 FR-003-B 为 P1 优先级）
- 复选框状态同步逻辑（空会话时禁用）

**补充需求**:

| 需求项 | 优先级 | 说明 |
|--------|--------|------|
| 最近会话获取 API 调用 | P1 | 勾选"继续会话"时调用 /api/sessions 获取最近会话 |
| 空会话处理 | P2 | 无历史会话时禁用复选框并提示 |
| 会话 ID 缓存 | P2 | 前端缓存最近会话 ID |

## 4. 验收标准

### 4.1 FR-001 验收标准

| 验收项 | 验收条件 |
|--------|---------|
| 同步执行 | POST /api/task 返回完整任务结果 |
| 流式执行 | POST /api/task/stream 实时推送消息 |
| SSE 消息类型 | 支持 text/thinking/tool_use/tool_result/error/complete/ask_user_question |
| 输入验证 | prompt 为空时返回 422 错误 |

### 4.2 FR-002 验收标准

| 验收项 | 验收条件 |
|--------|---------|
| 项目列表 | GET /api/projects 返回所有项目 |
| 会话列表 | GET /api/sessions 返回指定项目会话 |
| 消息历史 | GET /api/sessions/{id}/messages 返回完整消息链 |
| 分页 | 支持 page/limit 参数 |

### 4.3 FR-003 验收标准

| 验收项 | 验收条件 |
|--------|---------|
| 会话恢复 | resume 参数正确恢复历史会话 |
| 继续会话 | 勾选后自动填充最近会话 ID |

| 工作目录锁定 | 恢复历史会话时工作目录不可修改 |

### 4.4 FR-004 验收标准

| 验收项 | 验收条件 |
