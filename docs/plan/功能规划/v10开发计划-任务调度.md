## v10.0.0 - 任务调度核心功能

### 功能概述

任务调度功能提供 24/7 自动化任务执行能力，支持立即执行任务队列和基于 Cron 表达式的定时任务调度，适用于自动化代码检查、文档更新、测试运行等重复性任务。

## 技术架构

### 后端架构

```
app/
├── scheduler/
│   ├── __init__.py          # 模块导出
│   ├── models.py            # 数据模型
│   ├── config.py            # 配置
│   ├── scheduler.py         # 调度器
│   ├── executor.py         # 执行器
│   ├── storage.py           # 存储管理器
│   ├── storage_*.py         # 存储子模块
│   ├── cron_*.py           # Cron 解析
│   └── security_*.py        # 安全模块
└── routers/
    └── scheduler.py         # API 路由
```

### 前端架构

```
web/static/modules/scheduler/
├── index.js                 # 主模块
├── state.js                # 状态管理
├── api.js                  # API 调用
├── utils.js                # 工具函数
├── dialogs/                # 对话框组件
│   ├── index.js
│   ├── addTaskDialog.js
│   ├── addScheduledDialog.js
│   ├── taskDetailDialog.js
│   └── cronHelperDialog.js
└── renderers/              # 渲染器
    └── index.js
```
