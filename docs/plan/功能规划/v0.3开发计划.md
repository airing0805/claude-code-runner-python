**技术设计**: docs/plan/技术设计/环境信息展示-技术设计.md
使用专家不同的专家角色完成后面的子任务。
    - API 开发 → 高级后端专家
    - 前端开发 → 高级前端专家
    - 测试编写 → 测试工程师 / 高级测试工程师
    - 安全审查 → 安全专家
    - 文档更新 → 技术文档工程师
    - 整体把控 → 高级全栈专家
