# v0.3.4 - MCP 服务器管理

## 1. 功能概述

管理 Claude Code 的 MCP（Model Context Protocol）外部服务连接。用户可以通过浏览器配置和管理 MCP 服务器。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 配置 MCP 服务器 | 用户想添加新的 MCP 服务器连接 |
| 管理现有服务器 | 用户想查看、编辑、删除已配置的 MCP 服务器 |
| 查看连接状态 | 用户想了解各 MCP 服务器的在线/离线状态 |
| 查看可用工具 | 用户想查看 MCP 服务器提供的工具列表 |

## 3. 功能列表

### 3.1 服务器列表

- 查看已配置的 MCP 服务器
- 显示服务器名称、连接类型、状态

### 3.2 添加服务器

- 配置新的 MCP 服务器连接
- 支持 stdio 类型（本地命令）
- 支持 http/sse 类型（远程服务）

### 3.3 编辑服务器

- 修改现有服务器配置
- 启用/禁用服务器

### 3.4 删除服务器

- 移除服务器配置

### 3.5 连接状态

- 查看服务器在线/离线状态
- 查看服务器提供的工具数量
