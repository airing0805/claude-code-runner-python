## v12.0.0 (2026-03-11)

### 界面重构

#### 新增功能

##### 工作空间组合控件
- 工作空间输入控件升级为组合控件（下拉+输入）
- 支持下拉选择历史工作空间（最多保留 10 条）
- 支持手动输入新路径
- 切换工作空间自动创建新会话

##### 历史记录抽屉
- 历史记录以侧边抽屉形式展示
- 点击历史记录即可继续会话
- 自动恢复工作空间路径和消息历史
- 支持按项目分组显示会话

##### 新会话功能
- 点击新会话按钮清空当前表单
- 会话 ID 显示为"新会话"
- 未保存内容检查和确认对话框
- 工作空间路径保持不变

#### 修改功能

##### 移除 Tab 标签页
- 彻底移除 Tab 标签页系统及相关代码
- 从多会话并行显示改为单会话界面
- 移除继续会话复选框（历史记录抽屉替代）

##### 数据迁移
- 实现 v11 到 v12 数据自动迁移
- 支持数据格式转换（多 Tab 对象 → 会话数组）
- 自动去重处理
- 提供迁移进度提示和错误处理

#### 移除功能

- Tab 标签页系统（包括拖拽、排序、关闭功能）
- 继续会话复选框
- v11 localStorage 数据结构

#### Breaking Changes

- **移除 Tab 标签页功能**: 从多会话并行改为单会话模式
- **localStorage 结构变更**: 从 v11 的 `{tabs: [...], activeTabId: ...}` 变为 v12 的会话数组格式
- **自动迁移**: 首次加载时自动迁移 v11 数据，迁移后旧数据保留但不使用

#### 技术改进

- 重构状态管理，简化单会话状态
- 优化消息渲染器，移除 Tab 相关逻辑
- 新增 Migration 模块处理版本迁移
- 新增 WorkspaceCombo 组件
- 更新 Session 模块支持新会话功能

### 测试覆盖

- 单元测试：63 个测试用例全部通过
- 集成测试：完整流程验证通过
- 代码审查：CRITICAL 0 个，HIGH 1 个（已修复）

### 性能优化

- 会话列表查询使用元数据缓存

- 会话历史支持分页

### 文档更新

## v7.1.0 (2026-03-10)

### 新增功能

#### 日志标签页

- 新增独立日志标签页（正常日志/错误日志）
- 标签页状态保持（切换后返回保持选中状态）
- 日志行样式区分（stdout 白色/浅色，stderr 红色/粉色）

#### 实时日志

- 新增 SSE 实时日志推送
- 日志类型过滤（stdout/stderr/all）
- 自动滚动到底部
- 暂停/恢复滚动功能
- 30 秒心跳保持连接，防止超时

#### 历史日志

- 新增历史日志分页加载
- 时间范围过滤（start_time/end_time）
- 加载状态指示

#### 日志搜索

- 关键字搜索功能
- 正则表达式搜索
- 搜索结果高亮
- 搜索结果导航（上一条/下一条）

### API 端点

- `GET /api/scheduler/tasks/{task_id}/logs/normal` - 获取正常日志
- `GET /api/scheduler/tasks/{task_id}/logs/error` - 获取错误日志
- `GET /api/scheduler/tasks/{task_id}/logs/stream` - SSE 实时日志流
- `GET /api/scheduler/tasks/{task_id}/logs/search` - 搜索日志
- `GET /api/scheduler/tasks/{task_id}/logs/count` - 获取日志数量统计

### 性能优化

- 前端虚拟滚动渲染大量日志
- 历史日志分页加载，减少内存占用
- 日志缓冲刷新，提升实时日志性能

### 文档更新

- 新增任务调度 API 接口文档（日志增强部分）
- 更新版本发布说明
