## v9.0.0 (2026-03-06)

### 新增功能

#### 当前会话功能

- 新增会话管理模块 SessionManager
- 支持任务同步执行 (`POST /api/task`)
- 支持任务流式执行 (`POST /api/task/stream`)
- 支持会话恢复功能
- 支持问答交互 (`POST /api/task/answer`)

#### 会话历史功能

- 新增项目列表 API (`GET /api/session/projects`)
- 新增会话列表 API (`GET /api/session/sessions`)
- 新增消息历史 API (`GET /api/session/sessions/{id}/messages`)
- 新增问答记录 API (`GET /api/session/projects/{name}/questions`)
- 实现元数据缓存机制，优化查询性能

#### 前端功能

- 新增会话信息栏组件
- 新增消息渲染器（支持 text/thinking/tool_use/tool_result/complete/error）
- 新增 SSE 客户端管理
- 新增问答对话框组件

### 性能优化

- 会话列表查询使用元数据缓存
- 项目列表支持分页
- 会话历史支持分页

### 文档更新

- 新增 API 接口文档
- 新增版本发布说明
- 更新开发指南（v9.0.0 功能说明）
