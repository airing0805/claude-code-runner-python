# 架构设计文档

> Claude Code Runner 架构设计
> **版本**: v9.0.3
> **最后更新**: 2026-03-06

## 1. 系统概述

Claude Code Runner 是一个基于 FastAPI 的 Web 服务，封装了 Claude Code (claude-agent-sdk)，提供任务执行、会话管理、定时调度等功能。

### 1.1 核心特性

- **任务执行**: 同步/流式任务执行，支持会话恢复
- **会话管理**: 多会话并行管理，历史会话查询
- **任务调度**: 队列任务、定时任务（Cron）
- **Web UI**: 交互式 Web 界面
- **扩展能力**: MCP 服务器、插件系统、技能系统

## 3. 模块说明

### 3.1 后端模块

| 模块 | 路径 | 说明 |

## 4. 模块间交互

### 4.1 任务执行流程

```
Client Request
     │
     ▼
┌─────────────┐
│ Task Router │  POST /api/task or /api/task/stream
└──────┬──────┘
       │
       ▼
┌──────────────┐
│Claude Runner │  调用 claude-agent-sdk
│   Client    │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│Claude Code  │  Claude Code CLI 进程
│    CLI      │
└─────────────┘
       │
       ▼
    SSE Stream / Response
       │
       ▼
    Client
```

### 4.2 会话管理流程

```
创建会话 / 恢复会话
       │
       ▼
┌────────────────┐
│ Session Router │  POST /api/task/new-session
└───────┬────────┘
        │
        ▼
┌────────────────┐
│Session Manager │  管理内存中的活动会话
└───────┬────────┘
        │
        ▼
┌────────────────┐
│Claude Runner   │  创建/恢复 SDK 会话
└───────┬────────┘
        │
        ▼
   返回 session_id
```

### 4.3 任务调度流程

```
创建任务 / 定时任务触发
       │
       ▼
┌────────────────┐
│Scheduler Router│  POST /api/scheduler/tasks
└───────┬────────┘
        │
        ▼
┌────────────────┐
│   Scheduler    │  APScheduler 调度
└───────┬────────┘
        │
        ▼
┌────────────────┐
│   Executor     │  后台执行任务
│ (Thread Pool)  │
└───────┬────────┘
        │
        ▼
┌────────────────┐
│Claude Runner   │  执行实际任务
└───────┬────────┘
        │
        ▼
┌────────────────┐
│    Storage     │  存储任务结果
└────────────────┘
```

### 4.4 会话历史查询流程

```
查询历史 / 会话列表
       │
       ▼
┌────────────────┐
│ Session Router │  GET /api/session/sessions
└───────┬────────┘
        │
        ▼
┌────────────────┐
│  File System   │  读取 ~/.claude/projects/
└───────┬────────┘
        │
        ▼
   返回历史数据
```

## 6. 扩展性设计

### 6.1 认证扩展

系统支持多种认证方式，可通过配置启用：

- **无认证**: 允许所有请求
- **API Key**: 通过 `X-API-Key` header 传递
- **OAuth2**: 通过 `Authorization: Bearer <token>` 传递

认证配置位于 `app/auth/` 目录，可根据需求扩展新的认证 provider。

### 6.2 MCP 服务器扩展

系统支持动态加载 MCP (Model Context Protocol) 服务器：

```python
# 注册 MCP 服务器
from app.mcp.manager import mcp_manager

# 加载自定义 MCP 服务器
mcp_manager.register_server("my-server", {
    "command": "npx",
    "args": ["-y", "@myorg/mcp-server"]
})
```

### 6.3 技能系统扩展

支持自定义技能（Skills），用于扩展 Claude Code 的能力：

```python
# 定义技能
from app.skills.schemas import Skill, SkillDefinition

skill = Skill(
    name="文件处理",
    description="处理文件的高级操作",
    actions=[
        SkillDefinition(
            name="convert_encoding",
            description="转换文件编码"
        )
    ]
)
```

### 6.4 插件系统扩展

支持加载自定义插件，扩展核心功能：

```python
# 插件结构
# plugins/
#   my_plugin/
#     __init__.py
#     plugin.yaml
#     handler.py
```

### 6.5 Webhook 通知扩展

任务执行完成后，可通过 Webhook 发送通知：

```python
# 配置 Webhook
webhook_url = "https://your-server.com/notify"
webhook_events = ["task.complete", "task.error", "question.ask"]
```

### 6.6 自定义工具扩展

可在 `app/routers/` 中添加新的 Router 来扩展 API 功能：

```python
from fastapi import APIRouter

custom_router = APIRouter(prefix="/api/custom", tags=["Custom"])

@custom_router.get("/custom-endpoint")
async def custom_endpoint():
    return {"message": "Custom endpoint"}
```

## 8. 安全性考虑

### 8.1 工作目录限制

- 默认只能访问配置的工作目录
- 支持配置多个允许的工作目录
- 可通过 `workspace` 参数指定子目录

### 8.2 工具权限控制

- 可配置允许使用的工具列表
- 支持不同权限模式：`read`、`write`、`bypassPermissions`

### 8.3 API 认证

- 支持 API Key 认证
- 支持 OAuth2 Bearer Token
- 支持 IP 白名单

### 8.4 请求频率限制

- 可配置请求频率限制
- 防止恶意请求和资源耗尽

## 9. 性能优化

### 9.1 会话缓存

- 活动会话保存在内存中
- 支持会话元数据缓存
- 减少重复加载开销

### 9.2 连接池

- HTTP 客户端使用连接池
- SDK 连接复用

### 9.3 异步处理

- 使用 FastAPI 异步框架
- SSE 流式输出减少等待时间

### 9.4 任务队列

- 后台任务使用线程池执行
- 支持任务超时控制
