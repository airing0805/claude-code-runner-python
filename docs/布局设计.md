# 管理系统布局设计

## 布局结构

采用经典的「顶部导航 + 左侧菜单 + 右侧内容」布局：

```
┌─────────────────────────────────────────────────────────────┐
│  顶部栏 (Header)                                             │
│  Logo | 系统标题                    状态 | 用户信息 | 设置    │
├───────┬─────────────────────────────────────────────────────┤
│ 左侧  │  面包屑 / 页面标题                                     │
│ 菜单  ├─────────────────────────────────────────────────────┤
│       │                                                     │
│  会话 │                                                     │
│  历史 │              右侧内容区                               │
│  示例 │            (Content Area)                            │
│  设置 │                                                     │
│       │                                                     │
│  ───  │                                                     │
│  收起 │                                                     │
└───────┴─────────────────────────────────────────────────────┘
```

## 布局规格

| 区域 | 宽度/高度 | 说明 |
|------|-----------|------|
| 顶部栏 | 高度 48-56px | 固定高度，深色背景 |
| 左侧菜单 | 宽度 200-240px | 可收起为 48px 图标模式 |
| 右侧内容 | flex: 1 | 自适应剩余宽度 |
| 内边距 | 16-24px | 内容区四周留白 |

## 响应式适配

| 屏幕宽度 | 左侧菜单 | 说明 |
|----------|----------|------|
| >1440px | 240px | 完整菜单 |
| 1024-1440px | 200px | 标准菜单 |
| <1024px | 48px | 图标模式 |

## 关键特性

### 1. 菜单收起

- 收起按钮放在菜单**底部**
- 收起时显示图标，悬停显示 tooltip
- 状态持久化到 localStorage

### 2. 滚动行为

- 左侧菜单：固定不滚动（菜单项多时内部滚动）
- 右侧内容：独立滚动
- 顶部栏：固定不滚动

### 3. 内容区规范

- 所有右侧功能页面布局保持一致
- 内容区最小宽度：800px
- 内容区内边距：24px

## CSS 参考

```css
/* 整体布局 */
.app-layout {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

/* 顶部栏 */
.app-header {
  height: 56px;
  flex-shrink: 0;
}

/* 主体区域 */
.app-body {
  display: flex;
  flex: 1;
  overflow: hidden;
}

/* 左侧菜单 */
.app-sidebar {
  width: 240px;
  flex-shrink: 0;
  transition: width 0.3s;
}

.app-sidebar.collapsed {
  width: 48px;
}

/* 右侧内容 */
.app-content {
  flex: 1;
  overflow: auto;
  padding: 24px;
  min-width: 800px;
}
```
