# v0.3.8 - 钩子配置

## 1. 功能概述

管理 Claude Code 钩子（Hooks）配置，允许用户查看和配置钩子触发规则。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 查看钩子配置 | 用户想了解当前配置的钩子 |
| 配置钩子 | 用户想添加或修改钩子规则 |

## 3. 功能列表

### 3.1 钩子列表

- 查看所有已配置的钩子
- 显示钩子名称、类型、触发时机

### 3.2 钩子类型说明

- PreToolUse：工具执行前
- PostToolUse：工具执行后
- Stop：会话结束时
- SessionStart：会话开始时
- Notification：通知事件

### 3.3 钩子配置

- 添加新钩子
- 编辑现有钩子
- 删除钩子

## 4. 验收标准

- [x] 钩子列表正确显示
- [x] 钩子类型说明清晰
- [x] 可以添加/编辑/删除钩子
