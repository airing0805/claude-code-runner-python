# v0.3.5 - 技能系统管理

## 1. 功能概述

管理 Claude Code 的 Skill（技能）系统，允许用户查看、启用/禁用技能，配置技能参数。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 查看技能列表 | 用户想了解有哪些可用技能 |
| 启用/禁用技能 | 用户想启用或停用某些技能 |
| 查看技能详情 | 用户想了解技能的详细说明和参数 |
| 配置技能参数 | 用户想自定义技能的参数设置 |

## 3. 功能列表

### 3.1 技能列表

- 查看所有可用技能
- 显示技能名称、描述、状态
- 按类别筛选技能

### 3.2 技能详情

- 查看技能说明
- 查看参数配置
- 查看使用示例

### 3.3 启用/禁用

- 激活或停用技能

### 3.4 用户配置

- 保存用户偏好的技能设置

## 4. 验收标准

- [x] 技能列表正确显示
- [x] 技能可以按类别筛选
- [x] 可以启用/禁用技能
- [x] 技能详情正确显示
- [ ] 参数配置可以保存（功能预留，暂未实现）
