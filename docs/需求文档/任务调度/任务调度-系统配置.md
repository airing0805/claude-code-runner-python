# 任务调度-系统配置

## 1. 系统配置规范

### 1.1 核心配置参数

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `poll_interval` | 10000ms | 队列轮询间隔 |
| `max_retries` | 2 | 最大重试次数 |
| `task_timeout` | 600000ms | 单任务默认超时 |
| `max_history` | 1000 | 历史记录最大数量 |
| `data_dir` | data/ | 数据文件目录 |

### 1.2 CLI 插件配置 【待实现】

| 参数 | 类型 | 说明 |
|------|------|------|
| `command` | string | CLI 可执行命令 |
| `parameters.method` | 'stdin' \| 'flag' | 参数传递方式 |
| `parameters.flag` | string | 命令行参数标志 |
| `parameters.required` | boolean | 是否必填 |

### 1.3 Web界面配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `refresh_interval` | 5000ms | 页面自动刷新间隔 |
| `max_log_lines` | 1000 | 日志显示最大行数 |
| `theme` | "light" | 界面主题（light/dark） |
| `auto_scroll` | true | 日志自动滚动到底部 |

## 2. 非功能性需求

### 2.1 性能需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 任务响应延迟 | < 10s | 从任务添加到开始执行 |
| 最大并发任务 | 1 | 当前版本串行执行 |
| 内存占用 | < 1GB | 超过后自动重启 |
| 日志文件大小 | < 5MB | 轮转限制 |

### 2.2 可靠性需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 服务可用率 | 99.9% | FastAPI 异步架构保障 |
| 故障恢复时间 | < 10s | 自动重试机制 |
| 数据持久化 | 100% | JSON 文件存储 |

### 2.3 兼容性需求

| 平台 | 支持状态 |
|------|---------|
| Windows | 完全支持 |
| Linux | 完全支持 |
| macOS | 完全支持 |

### 2.4 安全性需求

| 需求 | 说明 |
|------|------|
| 用户认证 | 所有操作需要JWT认证 |
| 输入验证 | 前后端双重验证 |
| 目录限制 | 工作目录必须在允许范围内 |
| 敏感操作 | 需要二次确认 |
| 日志审计 | 记录所有关键操作 |

## 4. 术语表

| 术语 | 定义 |
|------|------|
| CLI | Command Line Interface，命令行界面 |
| Cron | Unix 系统的定时任务调度表达式 |
| FIFO | First In First Out，先进先出队列 |
| stdin | 标准输入流 |
| stdout | 标准输出流 |
| stderr | 标准错误流 |
| auto_approve | 自动批准，跳过人工确认 |
| workspace | 工作目录，任务执行的上下文路径 |
