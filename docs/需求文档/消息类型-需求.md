# 消息类型

## 消息类型定义

| 类型 | 说明 | 样式 |
|------|------|------|
| text | 文本内容 | 蓝色边框 |
| thinking | 思考过程 (v0.2.11+) | 黄色背景，可折叠 |
| tool_use | 工具调用 | 绿色边框 |
| tool_result | 工具结果 | 蓝色边框 |
| error | 错误信息 | 红色边框 |
| complete | 任务完成 | 蓝色边框 |
| info | 提示信息 | 天蓝色边框 |

## 3.1 历史消息渲染 (v0.2.11+)

- 支持多轮对话显示，每轮包含用户输入和 AI 响应
- 按轮次分组显示，每轮显示轮次编号
- 工具调用显示工具名称和参数
- 工具结果可区分成功/错误状态
- 思考过程默认折叠，点击可展开
