## 项目概述

Claude Code Runner 是 Claude Code 的 Web 服务封装，通过 FastAPI 提供 REST API 和 Web 界面来调用 Claude Code 执行编程任务。

### 核心目标

- 提供 Web 界面调用 Claude Code 执行任务
- 支持会话延续和历史记录管理
- 支持 SSE 流式输出实时显示执行过程
- 支持任务队列和定时任务调度
