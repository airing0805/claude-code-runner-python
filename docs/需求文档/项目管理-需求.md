# 项目管理

## 1. 功能概述

提供多项目管理工作区，用户可以管理多个工作目录下的 Claude Code 项目，查看项目列表和统计信息。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 查看所有项目 | 用户想了解有多少项目使用了 Claude Code |
| 项目统计 | 用户想了解每个项目的会话数量和使用的工具 |
| 项目导航 | 用户想快速切换到不同项目的工作区 |

## 3. 功能列表

### 3.1 项目列表

- 获取所有使用过 Claude Code 的项目
- 每个项目显示：
  - 编码名称（URL 友好格式）
  - 原始项目路径
  - 会话数量统计
  - 使用的工具列表汇总

### 3.2 项目信息

- 显示项目的完整信息：
  - 项目路径
  - 会话数量
  - 工具使用统计

### 3.3 项目编码规则

将文件路径转换为 URL 友好的编码格式：

| 原字符 | 编码后 |
|--------|--------|
| \ | - |
| / | - |
| : | 保留 |
| 空格 | - |

示例：
- `E:\workspaces\2026_python\project` → `E--workspaces-2026_python-project`
