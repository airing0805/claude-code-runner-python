# v0.3.7 - 插件管理

## 1. 功能概述

管理 Claude Code 插件，允许用户查看已安装的插件、启用/禁用插件。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 查看插件列表 | 用户想了解有哪些已安装的插件 |
| 启用/禁用插件 | 用户想启用或停用某些插件 |
| 查看插件详情 | 用户想了解插件的功能说明 |

## 3. 功能列表

### 3.1 插件列表

- 查看所有已安装的插件
- 显示插件名称、描述、状态

### 3.2 插件详情

- 查看插件功能说明
- 查看插件版本和作者

### 3.3 启用/禁用

- 激活或停用插件

## 4. 验收标准

- [ ] 插件列表正确显示
- [ ] 可以启用/禁用插件
- [ ] 插件详情正确显示
