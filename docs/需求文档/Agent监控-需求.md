# v0.3.6 - Agent 监控

## 1. 功能概述

监控和管理由主任务通过 Task 工具触发的子代理（Sub-Agent）。用户可以查看子代理状态、查看日志、终止失控的子代理。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 查看子代理列表 | 用户想了解当前运行的所有子代理 |
| 查看执行进度 | 用户想查看子代理的执行状态和进度 |
| 终止子代理 | 用户想手动终止运行中的子代理 |
| 查看日志 | 用户想查看子代理的执行日志 |

## 3. 功能列表

### 3.1 子代理列表

- 查看所有子代理及状态
- 按状态过滤（running、completed、terminated、failed）
- 按父任务过滤

### 3.2 子代理详情

- 查看执行进度
- 查看工具使用情况
- 查看文件变更

### 3.3 终止子代理

- 手动终止运行中的子代理

### 3.4 日志流

- SSE 流式查看子代理执行日志

## 4. 验收标准

- [ ] 子代理列表正确显示
- [ ] 可以按状态过滤
- [ ] 子代理详情正确显示
- [ ] 可以终止运行中的子代理
