## 1. 概述

### 1.1 定义

交互流程需求定义了用户与界面交互的完整路径，确保用户体验流畅、操作直观。

### 1.2 涉及流程

| 流程 | 说明 |
|------|------|
| 工作空间切换 | 选择或输入工作空间，创建新会话 |
| 历史记录继续 | 选择历史会话，加载到当前界面 |
| 新会话清空 | 点击新会话按钮，清空表单 |
| 工作空间历史 | 保存和显示最近使用的工作空间 |

## 2. 工作空间切换流程

### 2.1 流程图

```
用户操作                          系统处理
   │                                │
   │   点击工作空间下拉框            │
   ├──────────────────────────────► │
   │                                │
   │   选择历史工作空间或输入新路径  │
   ├──────────────────────────────► │
   │                                │
   │                                ├─ 验证路径有效性
   │                                │
   │                                ├─ 路径存在？
   │                                │   ├─ 是 ─► 创建新会话
   │                                │   │         ├─ 生成会话ID
   │                                │   │         ├─ 清空输出区
   │                                │   │         ├─ 清空输入区
   │                                │   │         └─ 更新会话状态
   │                                │   │
   │                                │   └─ 否 ─► 显示错误提示
   │                                │
   │   <──── 显示成功/错误提示 ───────┤
   │                                │
```

### 2.2 详细步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击工作空间控件 | 打开下拉列表或输入框 |
| 2 | 选择历史工作空间 | 从下拉列表中选择最近使用的路径 |
| 3 | 或手动输入新路径 | 直接在工作空间控件中输入新路径 |
| 4 | 路径验证 | 系统验证路径是否有效 |
| 5 | 创建新会话 | 生成新的会话 ID |
| 6 | 清空界面 | 清空输出显示区和输入区 |
| 7 | 更新状态 | 显示"已切换到 [路径]"提示 |

### 2.3 用户反馈

| 场景 | 反馈内容 |
|------|---------|
| 路径验证成功 | 绿色提示："已切换到 [工作空间路径]" |
| 路径验证失败 | 红色错误："工作目录不存在：[路径]" |
| 权限不足 | 红色错误："无权限访问目录：[路径]" |
| 切换成功 | 会话 ID 更新，界面清空 |

## 3. 历史记录继续会话流程

### 3.1 流程图

```
用户操作                          系统处理
   │                                │
   │   点击"历史记录"菜单项        │
   ├──────────────────────────────► │
   │                                │
   │                                ├─ 加载历史记录列表
   │                                │
   │                                ├─ 从右侧滑出抽屉
   │                                │
   │   <──── 显示历史记录 ─────────┤
   │                                │
   │   点击某个会话项              │
   ├──────────────────────────────► │
   │                                │
   │                                ├─ 显示加载指示器
   │                                │
   │                                ├─ 加载会话数据
   │                                │   ├─ 读取会话元数据
   │                                │   ├─ 读取消息历史
   │                                │   ├─ 读取工具配置
   │                                │   └─ 读取工作空间路径
   │                                │
   │                                ├─ 恢复界面状态
   │                                │   ├─ 恢复工作空间路径
   │                                │   ├─ 恢复消息历史
   │                                │   ├─ 恢复工具配置
   │                                │   └─ 更新会话 ID
   │                                │
   │                                ├─ 关闭抽屉
   │                                │
   │   <──── 显示成功提示 ─────────┤
   │                                │
```

### 3.2 详细步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击左侧菜单"历史记录" | 打开历史记录抽屉 |
| 2 | 显示历史记录列表 | 按项目分组显示会话 |
| 3 | 点击某个会话项 | 选中要继续的会话 |
| 4 | 显示加载指示器 | 在抽屉内显示加载动画 |
| 5 | 加载会话数据 | 从本地存储读取会话数据 |
| 6 | 恢复界面状态 | 将数据恢复到界面 |
| 7 | 关闭抽屉 | 滑动收起抽屉 |
| 8 | 显示成功提示 | "已恢复会话：[会话摘要]" |

### 3.3 用户反馈

| 场景 | 反馈内容 |
|------|---------|
| 抽屉打开 | 平滑滑动动画，遮罩层变暗 |
| 加载中 | 抽屉内显示圆形加载指示器 |
| 加载成功 | 绿色提示："已恢复会话"，抽屉自动关闭 |
| 加载失败 | 红色错误："会话加载失败"，抽屉保持打开 |
| 会话恢复完成 | 界面显示恢复的消息历史和工具配置 |

## 4. 新会话清空流程

### 4.1 流程图

```
用户操作                          系统处理
   │                                │
   │   点击"新会话"按钮            │
   ├──────────────────────────────► │
   │                                │
   │                                ├─ 检查当前会话状态
   │                                │
   │                                ├─ 有未保存内容？
   │                                │   ├─ 是 ─► 显示确认对话框
   │                                │   │         "当前会话未保存，是否确定新建会话？"
   │                                │   │         ├─ 确认 ─► 清空界面
   │                                │   │         └─ 取消 ─► 不操作
   │                                │   │
   │                                │   └─ 否 ─► 清空界面
   │                                │
   │                                ├─ 清空输出显示区
   │                                │
   │                                ├─ 清空输入区
   │                                │
   │                                ├─ 重置工具配置
   │                                │
   │                                ├─ 清空会话ID
   │                                │
   │                                │
   │   <──── 显示成功提示 ─────────┤
   │                                │
```

### 4.2 详细步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 点击"新会话"按钮 | 触发新建会话操作 |
| 2 | 检查当前会话状态 | 判断是否有未保存内容 |
| 3 | 显示确认对话框（可选） | 询问用户是否确认 |
| 4 | 用户确认 | 点击"确定"按钮 |
| 5 | 清空输出显示区 | 移除所有消息和内容 |
| 6 | 清空输入区 | 清空任务输入框 |
| 7 | 重置工具配置 | 恢复工具默认配置 |
| 8 | 清空会话 ID | 显示"新会话"或空值 |
| 9 | 显示成功提示 | "已创建新会话" |

### 4.3 用户反馈

| 场景 | 反馈内容 |
|------|---------|
| 点击按钮 | 按钮按下效果，轻微放大 |
| 确认对话框 | 模态对话框居中显示 |
| 确认后清空 | 界面平滑清空，动画过渡 |
| 清空成功 | 绿色提示："已创建新会话" |
| 取消操作 | 对话框关闭，界面不变 |

## 5. 工作空间历史记录流程

### 5.1 流程图

```
用户操作                          系统处理
   │                                │
   │   使用工作空间（切换或新输入） │
   ├──────────────────────────────► │
   │                                │
   │                                ├─ 记录使用时间和路径
   │                                │
   │                                ├─ 更新历史记录列表
   │                                │   ├─ 移除重复记录
   │                                │   ├─ 添加或更新记录
   │                                │   └─ 按时间倒序排序
   │                                │
   │                                ├─ 保存到本地存储
   │                                │
   │                                └─ 最多保留 10 条
   │                                │
   │   <──── 更新下拉列表 ──────────┤
   │                                │
```

### 5.2 详细步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 用户使用工作空间 | 通过下拉选择或手动输入 |
| 2 | 记录使用信息 | 保存路径和使用时间戳 |
| 3 | 检查重复 | 判断路径是否已在历史记录中 |
| 4 | 更新历史记录 | 新记录插入或旧记录更新时间 |
| 5 | 排序记录 | 按最后使用时间倒序排列 |
| 6 | 限制数量 | 超过 10 条时删除最旧的记录 |
| 7 | 保存到本地 | 使用 localStorage 存储记录 |
| 8 | 更新下拉列表 | 重新渲染工作空间下拉框 |

### 5.3 用户反馈

| 场景 | 反馈内容 |
|------|---------|
| 保存成功 | 无感知，下拉列表自动更新 |
| 下拉列表更新 | 新使用的路径显示在列表顶部 |
| 历史记录满 | 最旧的记录自动移除 |

## 6. 状态流转

### 6.1 会话状态

| 状态 | 说明 | 触发条件 |
|------|------|----------|
| **新会话** | 尚未执行任务的会话 | 点击新会话按钮 |
| **执行中** | 任务正在执行 | 提交任务后 |
| **已完成** | 任务执行完成 | 接收到 complete 消息 |
| **已恢复** | 从历史记录恢复的会话 | 加载历史会话后 |

### 6.2 状态流转图

```
                    新会话
                       │
          ┌────────────┴────────────┐
          │                         │
          │ 切换工作空间          │ 继续会话
          │                         │
          ▼                         ▼
       新会话                    已恢复
          │                         │
          │                         │
          │ 提交任务               │ 提交任务
          │                         │
          ▼                         ▼
       执行中                    执行中
          │                         │
          │                         │
          │ 任务完成               │ 任务完成
          │                         │
          └────────────┬────────────┘
                       ▼
                    已完成
                       │
          ┌────────────┴────────────┐
          │                         │
          │ 切换工作空间          │ 点击新会话
          │                         │
          ▼                         ▼
       新会话                    新会话
```

## 7. 异常处理

### 7.1 工作空间切换异常

| 异常类型 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 路径不存在 | 阻止切换，显示错误 | "工作目录不存在：[路径]" |
| 权限不足 | 阻止切换，显示错误 | "无权限访问目录：[路径]" |
| 路径格式错误 | 阻止切换，显示错误 | "路径格式不正确" |
| 系统错误 | 阻止切换，显示错误 | "工作空间切换失败，请重试" |

### 7.2 历史记录加载异常

| 异常类型 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 数据不存在 | 显示空列表，提示 | "暂无历史记录" |
| 数据损坏 | 显示错误，保留空状态 | "历史记录数据损坏" |
| 加载超时 | 显示错误，重试选项 | "加载超时，是否重试？" |
| 系统错误 | 显示错误，重试选项 | "加载失败，请重试" |

### 7.3 新会话清空异常

| 异常类型 | 处理方式 | 用户提示 |
|---------|---------|---------|
| 清空失败 | 显示错误，保留原状态 | "清空失败，请重试" |
| 状态重置失败 | 显示错误，部分回滚 | "部分状态重置失败" |
