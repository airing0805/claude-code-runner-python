## 二、组件设计

### 2.1 任务输入框

#### 设计规格

| 属性 | 值 |
|------|-----|
| 高度 | 120px（最小 4 行） |
| 最大高度 | 400px（超过后显示滚动条） |
| 边框 | 2px 实线，圆角 8px |
| 边框颜色 | 默认: #3b82f6 / 聚焦: #2563eb |
| 背景颜色 | 默认: #f8fafc / 聚焦: #ffffff |
| 文字颜色 | #1e293b |
| 字体大小 | 14px |
| 行高 | 1.6 |
| 内边距 | 16px |
| 占位符文本 | "输入你的任务描述..." |

#### 交互状态

| 状态 | 样式描述 |
|------|---------|
| **默认** | 浅灰边框，浅灰背景 |
| **聚焦** | 蓝色边框，白色背景，阴影效果 |
| **有内容** | 蓝色边框 |
| **禁用** | 灰色边框，灰色背景，半透明文字 |

### 2.2 权限模式下拉单选

#### 设计规格

| 属性 | 值 |
|------|-----|
| 宽度 | 180px |

| 字体大小 | 13px |

| 背景色 | #ffffff |

| 圆角 | 6px |
| 下拉箭头 | #64748b |

#### 权限模式选项

| 模式 | 说明 |
|------|------|

## 三、响应式设计

### 3.1 断点定义

| 断点名称 | 宽度范围 | 设备类型 |
|----------|-----------|---------|
| **Mobile** | < 640px | 手机竖屏 |
| **Tablet** | 640px - 1024px | 平板、手机横屏 |
| **Desktop** | > 1024px | 桌面显示器 |

### 3.2 移动端适配

#### 布局调整

| 元素 | Desktop | Mobile |
|------|---------|--------|
| **底部输入区** | 全宽横向布局 | 纵向堆叠，输入框高度增加 |
| **消息展示区** | 固定宽度 800px | 全宽，最大宽度限制 |
| **会话信息栏** | 单行显示 | 多行显示，每行一个信息项 |

#### 交互优化

| 操作 | Desktop | Mobile |
|------|---------|--------|
| **执行按钮** | 点击触发 | 触摸触发，增大点击区域（最小 44x44px） |
| **消息滚动** | 鼠标滚轮 | 手指滑动 |

### 3.3 平板端适配

| 元素 | 布局调整 |
|------|---------|
| **底部输入区** | 两列布局（左权限模式，右发送/停止按钮） |
| **消息展示区** | 固定宽度 600px，居中显示 |
| **会话信息栏** | 单行显示，关键信息优先 |

## 四、动画与过渡

### 4.1 交互动画

| 动画类型 | 时长 | 缓动函数 | 说明 |
|----------|------|---------|------|
| **按钮悬停** | 150ms | ease-out | 背景颜色过渡 |
| **输入框聚焦** | 200ms | ease-in-out | 边框和阴影过渡 |
| **消息淡入** | 300ms | ease-out | 透明度从 0 到 1 |

### 4.2 加载动画

#### 执行按钮加载状态

```
[ 执行任务  ⏳ ]
         ↑ 旋转的 loading 图标
```

| 属性 | 值 |
|------|-----|
| 图标 | ⏳ |
| 旋转动画 | 360° / 1s，线性旋转 |
| 按钮状态 | 禁用，加载中文字颜色变暗 |

#### SSE 消息流动画

| 消息类型 | 动画效果 |
|----------|---------|
| **text** | 从底部向上滑入（100ms） |
| **thinking** | 淡入 + 呼吸效果（150ms） |
| **tool_use** | 左侧滑入 + 工具图标弹跳（200ms） |
| **tool_result** | 右侧滑入（200ms） |
| **complete** | 绿色边框闪烁（500ms x 3） |

### 4.3 页面过渡动画

| 过渡类型 | 效果 |
|----------|------|
| **新建会话** | 当前内容淡出 + 新输入框淡入（200ms） |
| **历史恢复** | 加载覆盖层 + 内容淡入（300ms） |
| **任务执行** | 按钮切换为停止状态（150ms） |

## 六、可访问性设计

### 6.1 键盘导航

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 在输入框和配置项间切换焦点 |
| `Shift + Tab` | 反向切换焦点 |
| `Enter` | 提交任务（当焦点在输入框） |
| `Esc` | 清空输入框，取消配置展开 |
| `Ctrl + K` | 聚焦任务输入框 |
| `Ctrl + H` | 打开历史记录 |
| `Ctrl + N` | 新建会话 |

### 6.2 屏幕阅读器支持

| ARIA 属性 | 用途 |
|-----------|------|
| `aria-label` | 为图标按钮提供文字描述 |
| `aria-expanded` | 标识折叠/展开状态 |
| `aria-disabled` | 标识禁用状态 |
| `role="status"` | 状态提示区域 |
| `role="log"` | 消息展示区域 |
| `aria-live="polite"` | 实时消息区域 |

### 6.3 视觉辅助

| 辅助特性 | 实现方式 |
|----------|---------|
| **高对比度模式** | 通过 CSS 变量支持切换 |
| **字体大小调整** | 支持用户设置 12px/14px/16px 三档 |
| **焦点指示** | 明显的焦点边框和阴影 |
| **颜色盲友好** | 状态不单靠颜色区分，配合图标和文字 |
