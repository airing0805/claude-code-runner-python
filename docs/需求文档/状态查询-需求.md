# 状态查询

## 1. 功能概述

提供基础服务状态和可用工具的查询功能，让用户可以快速了解服务运行状态和 Claude Code 的能力。

## 2. 用户场景

| 场景 | 描述 |
|------|------|
| 检查服务状态 | 用户想确认服务是否正常运行 |
| 查看工作目录 | 用户想确认当前配置的工作目录 |
| 查看活跃任务 | 用户想了解当前有多少任务在执行 |
| 查看可用工具 | 用户想了解 Claude Code 支持哪些工具 |

## 3. 功能列表

### 3.1 服务状态

- 获取当前服务运行状态
- 显示信息：
  - 服务状态（running/stopped）
  - 当前工作目录
  - 活跃任务数量

### 3.2 工具列表

> 工具列表功能已扩展到 [工具统计展示-需求.md](./工具统计展示-需求.md)，包含更详细的使用统计和权限模式说明。此处仅保留基础工具列表查询。

- 获取 Claude Code 可用的所有工具
- 显示每个工具的基本信息：
  - 工具名称
  - 功能描述

### 3.3 Web 界面

- 提供 Web 主页供用户访问
- 主页显示服务状态和快速入口
