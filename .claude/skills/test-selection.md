# Test Selection Skill

## Description
A simple skill to test ask_user_question functionality with multiple choice selection.

## When to use
When you need to present multiple options to the user and let them choose one.

## On invoke
Ask the user to select an option from the following choices:

1. **Option A** - First choice
2. **Option B** - Second choice
3. **Option C** - Third choice

Use the ask_user_question tool to present these options to the user in a multiple_choice format.
